# Food Tracking App Instructions

## About
Built almost entirely using Claude Code, no warrenty expressed or implied.

## Airtable
Airtable access token is in the file airtable.txt, along with the base name

## Application design
The goal is to duplicate the basic functionality of the airtable base with a PHP/SQL Lite application I can put on my webserver. 

No account management is needed at this time. 

The application should have a clean and modern interface, work well on a mobile phone, and should prioritize ease of data entry. 

The application should have the ability to export data as a csv. 

The application should be allow entry of new foods and meals with values. 

At this time, any data about food will be manually entered.
